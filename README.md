## About 📖

### Description
react native app that uses axios to make requests to a REST api that returns data from a Postgresql database, in production (heroku).
you can find the backend for this android app here: 
https://github.com/stelianok/Book-Api
### Objective
The project was developed with the intention to improve my knowledge about react,react native, react navigation and also learn
how to use my own backend using a Postgresql relational database.

---

## Installation 👷‍♂️

### Installing dependencies
inside the folder mobile, run:
`yarn` or `npm install`
### Running the app on Android, in development mode:
`yarn run android` or `react-native run-android`
### Starting the development server
`yarn start` or `react-native start`

---

## Features 🛠✨

- User can see a list of books
- User can add a new book
- use can use a bottom tab navigation.
